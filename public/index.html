I miss those O.G. links!

Need your short links converted to the original long links?


Input: I can take your .CSV with post ID's and short links and any other information. (Don't bother deleting extra columns).
Output: I'll give you a .CSV with the original ID's, shortened links, and extended link.

There are a few stipulations:
  1. Only takes single .CSV File at a time
  2. The Shortened Link Column must have the title "Link".
  3. The ID Column must be entitled "Channel Post ID".
  4. The program knows that some link data is not perfect and will correctly extract link if there is a space or comma after the link. Other characters might cause an issue.
  5. This is an 80% App(Pareto's Law) and will likely not convert every link. A time limit has been set, so the program will not spend an infinite amount of time on one link. LinkedIn links are known to elicit an error.
  6. If no link is in the "Link" column you will see "No Link". If the link couldn't be found, you will see "error."
If all goes according to plan:
  1. Click 'Choose File' and find your file.
  2. Give it some time, and wait for a button that says "Download File."
  3. If "Error" appears, something has failed.
  4. In that case feel free to reach out to Katharine Wagner (kwagner@spredfast.com).
